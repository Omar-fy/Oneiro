# Oneiro

Oneiro is a 2D scrolling platformer inspired by dreams, storytelling, and level traversal. The game follows Rue, a teen navigating a limbo between life and death, as players explore a fluid, dream-like world filled with challenges, shortcuts, and multiple endings. The narrative focuses on character development and symbolism, conveying emotion through dialogue, animation, and music.

## Features

2D scrolling platformer with dream-inspired level design

Multi-ending structure with hidden paths and shortcuts

Character-driven story with narrative depth and symbolism

Original sprite work and animations

Custom soundtrack designed to complement level pacing

## Development Highlights

Built in Unity using C# scripting for gameplay mechanics

Characterization inspired by narrative techniques from literature and game studies

Sprites created in Piskel with attention to visual storytelling

Music composed using Chiptune techniques to enhance immersion and rhythm-based exploration

Focus on creating a cohesive, multi-dimensional experience despite 2D limitations

## Learning Outcomes

Through the development of Oneiro, I gained experience in:

Unity game development and C# programming

Level design and story integration in 2D gameplay

Music composition and sound design for interactive media

Time management, problem solving, and iterative development

Professional workflow for combining visuals, music, and gameplay
